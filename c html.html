HTML (HyperText Markup Language)

Reference & Learning Portal — complete tag list + tutorials

Overview

HTML (HyperText Markup Language) is the foundation of web development. It provides the structure of web pages using tags and elements. Paired with CSS (for presentation) and JavaScript (for interactivity), HTML forms the core of modern websites and web apps.

HTML File Structure (Typical)

Below is a minimal valid HTML document and what the parts mean.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width,initial-scale=1">
    <title>My Page</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Hello world</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
  • <!DOCTYPE html> — declares HTML5 document type.
  • <head> — metadata, title, links to styles and scripts.
  • <body> — visible page content.

Common HTML Elements (with examples)

Headings & Text

<h1> to <h6>, <p>, <span>, <strong>, <em>.

<h1>Title</h1>

History & Timeline of HTML

  • 1991: Tim Berners-Lee creates the first HTML.
  • 1996: HTML 2.0 and later HTML 3/4 introduce more tags and structure.
  • 1999: Widespread adoption of HTML 4.
  • 2014 - present: HTML5 becomes the modern standard (semantic tags, multimedia, APIs).

HTML evolved from simple markup to a powerful platform enabling multimedia and web applications.

Semantic HTML Tags (Why they matter)

Semantic tags communicate meaning to browsers, search engines, and assistive tech. Use them for better accessibility and SEO.

  • <header> — page or section header
  • <nav> — navigation links
  • <main> — main content
  • <article> — self-contained article
  • <aside> — sidebar or tangential content
  • <footer> — footer content

HTML5: Modern Features & Examples

  • Multimedia: <video> and <audio> tags (no plugins needed).
  • Canvas: <canvas> for 2D/3D drawing with JavaScript.
  • Form improvements: new input types and validation.
  • APIs: Geolocation, Web Storage, Web Workers, Service Workers, etc.

Practical Uses & Importance of HTML

  • Creates page structure (headings, paragraphs, images, links).
  • Forms the base for websites, blogs, web apps, and documentation.
  • Essential for accessibility and SEO when used semantically.
  • Used in email templates, documentation, and hybrid mobile apps (via WebViews).
